## 🌟 The Vision
**GovSecAI** is not just a portal; it is the digital nervous system of modern governance. Designed to bridge the gap between citizens and authorities, it streamlines public concern management across three critical sectors: **Infrastructure (Roads)**, **Healthcare**, and **Financial Security**. 

By leveraging real-time data persistence and a high-fidelity analytics engine, GovSecAI ensures that every citizen's voice is heard, tracked, and resolved with professional efficiency.

---

## 💎 Core Architecture

### 📊 1. The Command Center (Admin Dashboard)
A high-impact executive dashboard providing 360-degree situational awareness.
- **Dynamic Analytics**: Real-time trend visualization using Chart.js.
- **Zonal Intelligence**: Drill down into specific cities and areas to identify hotspots.
- **Incident Management**: Direct control over area statuses (Pending, Resolving, Resolved).
- **Auto-Sync**: Background polling ensures data is never stale.

### 🏛️ 2. The Citizen Gateway (User Portal)
A premium, resident-facing interface built with accessibility and speed in mind.
- **Multi-Category Submission**: Tailored forms for Road issues, Health concerns, and Banking fraud.
- **Automated Metadata**: Intelligent generation of Complaint IDs, timestamps, and geolocation tags.
- **Zero-Friction UX**: Clean, intuitive design for all age groups.

### 🔍 3. The Transparency Engine (Status Tracking)
Building trust through real-time feedback.
- **Instant Search**: Direct access to investigation status using unique reference IDs.
- **Session History**: Localized tracking so users don't have to remember their IDs.
- **Live Updates**: Reflects administrative changes the moment they happen.

## 🚀 Quick Start Guide

### 1. Prerequisites
- Python 3.10+
- `pip install fastapi uvicorn pandas pydantic`

### 2. Launch the Ecosystem
```bash
# Navigate to the project root
python api/main.py
```
*The system will automatically initialize calculations and launch the browser interfaces.*

### 3. Access Points
- **Admin Dashboard**: `http://localhost:8000/dashboard/index.html`
- **Citizen Portal**: `http://localhost:8000/user/index.html`
